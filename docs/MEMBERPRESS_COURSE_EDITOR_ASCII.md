## Visual Elements Description

### Header
- MemberPress logo and branding
- Course title in header
- Navigation tabs: Course Page (active), Curriculum, Resources, Settings, Certificate

### Toolbar
- Left side: Add block (+), undo/redo arrows, view options
- Center: Breadcrumb showing "Kick It Up ... · Course" with command palette (⌘K)
- Right side: Save draft, device preview, copy, Publish button

### Main Content Area (Left)
- Large course title: "Kick It Up a Notch: Cooking Like Emeril"
- Gray "Classic" category badge
- Course description paragraph

### Sidebar (Right)
- Tab switcher: Course (active) | Block
- Document settings panel:
  - Featured image upload area
  - Excerpt field
  - Document stats (31 words, 1 minute read)
  - Last edited timestamp
  
### Metadata Fields
- **Status**: Draft (selected)
- **Publish**: Immediately
- **Slug**: kick-it-up-a-notch-cooking-like-emeril
- **Author**: seth
- **Template**: Single Posts

### Additional Sections
- Move to trash button (red text)
- Course Categories (collapsible)
- Course Tags (collapsible)
- Page Options (expanded with template selector)

### Design Notes
- Clean WordPress block editor interface
- MemberPress custom post type for courses
- Standard WordPress publishing workflow
- Responsive sidebar with course-specific metadata
