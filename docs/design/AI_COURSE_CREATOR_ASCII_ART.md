- `🎓` - AI Course Creator logo
- `✅` - Status indicator (Ready)
- `📋` - Previous Conversations button
- `┌─┐` - Icon buttons (+ for new, ↻ for history)
- `⋮⋮` - Drag handles for reordering
- `✏️` - Edit button
- `🗑️` - Delete button
- `→` - Send button arrow

## Interface Description:
- **Left Panel**: AI Assistant chat interface with conversation history
- **Right Panel**: Course Preview showing the structured course content
- **Header**: Purple gradient header with status indicator
- **Sections**: Collapsible course sections with lessons
- **Lessons**: Individual lesson items with duration and action buttons (on hover)
